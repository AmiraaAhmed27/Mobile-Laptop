# 📊 Mobile & Laptop Sales Dashboard

## 📌 Project Overview
Interactive Power BI dashboard designed to analyze mobile and laptop sales performance across different regions, brands, and specifications.

The dashboard provides insights into:
- Total Revenue
- Quantity Sold
- Average Price
- Brand Performance
- Product Distribution
- Core Specifications Analysis
- Sales Trends Over Time

---

## 🚀 Tools & Technologies
- Power BI
- SQL
- Excel
- DAX

---

## 📈 Key Insights
- Total Revenue reached **28.28bn**
- Total Quantity Sold exceeded **276K**
- Apple and Acer generated the highest revenue
- Laptop and Mobile sales contributed equally
- Revenue trends remained stable across 2023–2025
